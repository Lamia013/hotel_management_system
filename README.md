# Hotel Management System

A simple hotel management system that handles user registration, room management, and bookings.

## Live Demo

Try it online: https://midnights.infinityfreeapp.com

## Features

- User registration and login
- Room management
- Booking system
- Admin dashboard

## Running Locally

1. Place the project folder inside XAMPP’s `htdocs` folder.
2. Create the database using SQL file into phpMyAdmin.
3. Update the database connection in `connect.php` if needed.
4. Access the project at:
   http://localhost/hotel_ms/index.php
